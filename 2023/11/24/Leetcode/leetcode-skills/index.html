二维矩阵转一维向量

  • 二维转一维

    m 行 n 列矩阵：vector<vector<int>> graph2d(m, vector<int> (n))

    可以转成：vector<int> graph1d(m * n)

    访问元素： graph2d[x][y] --> graph1d[x * n + y]

  • 一维转二维

    x = v / n y = v % n

    graph1d[v] --> ``graph2d[v / n][v % n]`
